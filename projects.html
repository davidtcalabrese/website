ISWAM

The IoT Severe Weather Alert Monitor. A full stack JavaScript/Node application that calls APIs for current weather data and NWS alerts and sends them to the browser and an IoT device. No live version on GitHub pages.

Using JavaScript and chart.js, this tax calculator tool allows users to enter expected income and see how much of it will end up going to Uncle Sam as well as a breakdown of where the different taxes are going.
